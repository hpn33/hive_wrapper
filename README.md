Hive Wrapper

A Wrapper & Util for Hive


## Fundations

* Box
* Box Static Properties
* Box Type Properties
* Box migration



## How use
on pubspec.yaml add this
```
    hive_wrapper:
        git:
            url: https://github.com/hpn33/hive_wrapper.git
```

for new update just run flutter pub upgrade
